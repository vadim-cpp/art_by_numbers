# ArtByNumbers - Преобразование фотографий в раскраски по номерам

Профессиональное desktop-приложение на PyQt для автоматического преобразования любых фотографий в раскраски по номерам с использованием современных алгоритмов компьютерного зрения и машинного обучения. Приложение поддерживает интуитивный drag-and-drop интерфейс, предоставляет множество методов квантования цветов (KMeans, Median Cut, Frequency Smooth), различные алгоритмы выделения контуров и автоматическую нумерацию областей, позволяя создавать качественные схемы для художественного творчества как для начинающих, так и для опытных художников.
